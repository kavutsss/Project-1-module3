# Project-1-module3

## Project Manager

A simple portfolio app built to ensure the projects of the company are managed efficiently.

## Features

-Add project's
-Search through projects
-Delete projects
-Responsive design
-Modern UI

## Tech Stack

-JavaScript
-React
-tailwindcss
-Vite
-Jest

## Installation

Clone repo
```
git clone https://github.com/kavuts/Project-1-module3.git

```

Install dependencies
```
npm install
```
# Firebase setup

 -Go to Firebase Console
 -Create a new project
-Enable Firestore
 -Copy config to src/firebase.js

## Run the app
```
npm run dev
```
## Usage

-Add your projects
-Search through projects
-Delete projects

## Testing

```
npm run test
```
## Deployment

-Deploy to Firebase and GitHub Actions
Deployed at: https://project-1-module3.web.app
